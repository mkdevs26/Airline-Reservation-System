# Airline Reservation System
A console-based Airline Reservation System developed in C++ for Programming Fundamentals lab.
## Features
- Role-based access (Admin & Passenger)
- Flight management (add, remove, view)
- Passenger booking and cancellation
- Seat management with visual layout
- Booking history and reporting
- Seat management
- Persistent data storage

## Concepts Used
- Functions
- Arrays
- Structures
- Pointers
- Dynamic Memory
- File Handling
- Conditional Statements
- Loops
